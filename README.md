# El Pollo Loco — Jump’n’Run Game

**El Pollo Loco** ist ein kleines, aber ernst gemeintes **Jump’n’Run** im Browser. Du spielst **Pepe**, sammelst **Tabasco-Flaschen**, weichst Gegnern aus oder räumst sie ab – und arbeitest dich Schritt für Schritt zum Endboss vor: einem **riesigen, aggressiven Huhn**.

Entwickelt mit **HTML, CSS und Vanilla JavaScript** – **objektorientiert** aufgebaut.

---

## Gameplay

- Sammle **Tabasco-Flaschen**, um Gegner aus der Distanz abzuwerfen.
- Weiche **Küken** und **Hühnern** aus, um keinen Schaden zu nehmen.
- Alternativ: **Spring den Gegnern auf den Kopf**, um sie zu besiegen.
- Wirst du getroffen, verlierst du **Energie**.
- Nach Treffern durch Küken/Hühner kannst du **Münzen sammeln**, um **Energie zu heilen**.
- Ziel: Überlebe den Weg bis zum **Endboss** und besiege ihn.

---

## Features

- **Jump’n’Run Core Loop**: laufen, springen, ausweichen, kämpfen
- **Wurfmechanik**: Tabasco-Flaschen als Projectile Weapon
- **Stomp-Mechanik**: Gegner durch Draufspringen besiegen
- **Energie-System**: Treffer reduzieren Health/Energy
- **Healing über Coins**: Münzen sammeln, um Energie zu regenerieren
- **Boss Fight**: Endgegner am Ende des Levels
- **OOP Architektur**: Entities/Logik in Klassen strukturiert
- **Pure Web Tech**: HTML, CSS, JavaScript (keine Frameworks)
